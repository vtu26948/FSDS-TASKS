Automated Logging System

Using Triggers & Views (Audit Logging)

Trigger Function

A database trigger automatically logs every INSERT or UPDATE operation. This helps maintain audit trails in enterprise systems. 

CREATE TRIGGER log_changes
AFTER INSERT OR UPDATE ON users
FOR EACH ROW
INSERT INTO audit_log(user_id, action, timestamp)
VALUES (NEW.id, 'INSERT/UPDATE', NOW());

Daily Activity View

A view is created to display daily activity reports. 

CREATE VIEW daily_report AS
SELECT DATE(timestamp) AS date,
COUNT(*) AS total_actions
FROM audit_log
GROUP BY DATE(timestamp);
